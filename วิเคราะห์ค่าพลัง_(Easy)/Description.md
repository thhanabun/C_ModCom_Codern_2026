# Red_Release_(Easy)

คุณคือผู้ครอบครอง ดวงตาแห่งการวิเคราะห์ และคุณได้รับการมอบหมายใหัทำโปรเจค Red Release
โดยมีดวงตาที่สามารถมองเห็นค่าพลังของผู้เล่นได้ จะมีพลังอยู่สามอย่าง ได้แก่

พลังการอ่านเกม (Vision): 0-30 คะแนน
พลังการเล่นเป็นทีม (Teamwork): 0-30 คะแนน
พละกำลัง (Physical): 0-40 คะแนน

รับค่าพลังทั้ง 3 อย่าง นำมารวมกัน และประเมินระดับศักยภาพของผู้เล่นตามเกณฑ์ต่อไปนี้:

คะแนนรวม 0 - 60: แสดงผลว่า 'Normal Player'
คะแนนรวม 61 - 80: แสดงผลว่า 'Good Player'
คะแนนรวม 81 - 100: แสดงผลว่า 'Talented Player'
คะแนนรวมมากกว่า 100 หรือมีค่าเป็นลบ: แสดงผลว่า 'Something went wrong'

## ข้อมูลนำเข้า (Input)

| ลำดับ | รายละเอียด |
| :--- | :--- |
| บรรทัดที่ 1 | ชื่อ นามสกุล |
| บรรทัดที่ 2 | จำนวนเต็ม a แทนค่าพลัง โดยที่ $0 \le a \le 30$|
| บรรทัดที่ 3 | จำนวนเต็ม b แทนค่าพลัง โดยที่ $0 \le b \le 30$ |
| บรรทัดที่ 4 | จำนวนเต็ม c แทนค่าพลัง โดยที่ $0 \le c \le 30$ |

## ข้อมูลส่งออก (Output)

| ลำดับ | รายละเอียด |
| :--- | :--- |
| บรรทัดที่ 1 | เกณฑ์ที่ประเมณออกมา |
