# เหลืองอ่อนหรือแก่\_(Easy)

ทุกคนคงทราบกันดีว่าในเกมฟุตบอล หากนักเตะทำผิดกฎเช่น Zidane เอาหัวโหม่ง Chiellini นักเตะจะได้รับโทษเป็นใบสี (เหลือง และ แดง) โดย FIFA ทราบถึงปัญหานี้ดีว่าการตัดสินว่านักเตะจะได้รับใบใดนั้นขึ้นอยู่กับความรุนแรงของการกระทำ FIFA จึงต้องการระบบคำนวณว่าการกระทำนั้นเป็นใบเหลืองหรือใบแดง โดย

- คะแนนความรุนแรงมากกว่า 7 คือใบแดง
- คะแนนน้อยกว่าหรือเท่ากับ 7 คือใบเหลือง

## ข้อมูลนำเข้า (Input)

| **ลำดับ**   | **รายละเอียด**                                                  |
| ----------- | --------------------------------------------------------------- |
| บรรทัดที่ 1 | จำนวนเต็ม `s` แทนคะแนนความรุนแรงของการกระทำ โดยที่ 0 <= s <= 10 |

## ข้อมูลส่งออก (Output)

| **ลำดับ**   | **รายละเอียด**                                                                    |
| ----------- | --------------------------------------------------------------------------------- |
| บรรทัดที่ 1 | ใบที่ผู้เล่นได้รับ (หากใบแดงให้แสดง "RED CARD", หากใบเหลืองให้แสดง "YELLOW CARD") |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
| ------------------------ | ------------------------ |
| 7                        | YELLOW CARD              |
| 9                        | RED CARD                 |
