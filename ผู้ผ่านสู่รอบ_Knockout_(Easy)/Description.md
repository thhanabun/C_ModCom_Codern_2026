# ผู้ผ่านเข้าสู่รอบ_Knockout\_(Easy)

หลังจากจบการแข่งขันรอบแบ่งกลุ่มของฟุตบอลโลก 2026 ทีมที่มีคะแนนมากกว่า 6 คะแนนขึ้นไปจะผ่านเข้าสู่รอบสุดท้าย โดย

points = wins \* 3 + draws

## ข้อมูลนำเข้า (Input)

| **ลำดับ**   | **รายละเอียด**                                                 |
| ----------- | -------------------------------------------------------------- |
| บรรทัดที่ 1 | จำนวนเต็ม `w` แทนจำนวนเกมการแข่งขันที่ชนะ โดยที่ 0 <= w <= 10  |
| บรรทัดที่ 1 | จำนวนเต็ม `d` แทนจำนวนเกมการแข่งขันที่เสมอ โดยที่ 0 <= d <= 10 |

## ข้อมูลส่งออก (Output)

| **ลำดับ**   | **รายละเอียด**                 |
| ----------- | ------------------------------ |
| บรรทัดที่ 1 | สถานะการผ่านเข้ารอบ (0 หรือ 1) |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
| ------------------------ | ------------------------ |
| 3 0                      | 1                        |
| 1 0                      | 0                        |
