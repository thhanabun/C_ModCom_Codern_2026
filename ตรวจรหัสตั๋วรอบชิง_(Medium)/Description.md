# ตรวจรหัสตั๋วรอบชิง_(Medium)

รอบชิงของฟุตบอลโลกมีการตรวจรหัสตั๋วเข้าชม รหัสตั๋วแต่ละใบมีความยาว 6 ตัวอักษร และจะถือว่าใช้ได้เมื่อผ่านเงื่อนไขทั้งหมดต่อไปนี้

1. ตัวแรกเป็น `F`
2. ตัวที่สองเป็นตัวอักษรภาษาอังกฤษตัวพิมพ์ใหญ่ `A` ถึง `Z` แทนโซนที่นั่ง
3. ตัวที่สามถึงตัวที่หกเป็นตัวเลขทั้งหมด
4. หมายเลขที่นั่งจากตัวเลข 4 หลักต้องอยู่ในช่วง 1 ถึง 9999
5. ผลรวมของตัวเลข 4 หลักต้องหารด้วย 3 ลงตัว

ให้รับรหัสตั๋วหลายใบ แล้วสรุปจำนวนรหัสที่ใช้ได้ จำนวนรหัสที่ใช้ไม่ได้ โซนที่มีรหัสใช้ได้มากที่สุด และหมายเลขที่นั่งมากที่สุดของรหัสที่ใช้ได้

หากมีหลายโซนที่มีจำนวนรหัสใช้ได้เท่ากัน ให้เลือกโซนที่ตัวอักษรน้อยกว่า หากไม่มีรหัสที่ใช้ได้ ให้แสดงโซนเป็น `-` และหมายเลขที่นั่งมากที่สุดเป็น 0

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `n` แทนจำนวนรหัสตั๋ว โดยที่ 1 <= n <= 100 |
| บรรทัดที่ 2 ถึง n + 1 | สตริง `code` ความยาว 6 แทนรหัสตั๋วแต่ละใบ |

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนรหัสที่ใช้ได้ ตามรูปแบบ `Valid: x` |
| บรรทัดที่ 2 | จำนวนรหัสที่ใช้ไม่ได้ ตามรูปแบบ `Invalid: y` |
| บรรทัดที่ 3 | โซนที่มีรหัสใช้ได้มากที่สุด ตามรูปแบบ `Top zone: z` |
| บรรทัดที่ 4 | หมายเลขที่นั่งมากที่สุดของรหัสที่ใช้ได้ ตามรูปแบบ `Max seat: s` |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 5<br>FA0003<br>FB0012<br>FA1110<br>FC1234<br>XA0003 | Valid: 3<br>Invalid: 2<br>Top zone: A<br>Max seat: 1110 |
| 3<br>FA0001<br>FB0000<br>F10003 | Valid: 0<br>Invalid: 3<br>Top zone: -<br>Max seat: 0 |
