# ป้ายสกอร์เริ่มเกม_(Easy)

ก่อนเริ่มการแข่งขันฟุตบอลโลกจำลองของ CPE 40 ป้ายสกอร์ของสนามต้องแสดงสถานะเริ่มเกมและสกอร์เริ่มต้น

ให้เขียนโปรแกรมภาษา C ที่แสดงข้อความตามที่กำหนด โดยใช้ `printf()`

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| - | ไม่มีข้อมูลนำเข้า |

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | แสดงข้อความ `CPE 40 Match Start` |
| บรรทัดที่ 2 | แสดงข้อความ `Score 0 - 0` |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| - | CPE 40 Match Start<br>Score 0 - 0 |
