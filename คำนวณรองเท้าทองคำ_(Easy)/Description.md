# คำนวณรองเท้าทองคำ\_(Easy)

เมื่อฟุตบอลโลก 2026 ได้เริ่มขึ้น ประธานเป้ต้องการระบบคำนวณคะแนนของตัวเองสำหรับรางวัล Golden Boot โดยกำหนดว่า

- ยิงประตู = 4 คะแนน
- แอสซิสต์ = 1 คะแนน

## ข้อมูลนำเข้า (Input)

| **ลำดับ**   | **รายละเอียด**                                      |
| ----------- | --------------------------------------------------- |
| บรรทัดที่ 1 | จำนวนเต็ม `g` แทนจำนวนประตู โดยที่ 0 <= g <= 100    |
| บรรทัดที่ 2 | จำนวนเต็ม `a` แทนจำนวนแอสซิสต์ โดยที่ 0 <= a <= 100 |

## ข้อมูลส่งออก (Output)

| **ลำดับ**   | **รายละเอียด**                     |
| ----------- | ---------------------------------- |
| บรรทัดที่ 1 | คะแนนรางวัล Golden Boot ของผู้เล่น |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
| ------------------------ | ------------------------ |
| 5<br>3                   | 23                       |
