# ระยะทาง_(Easy)

บนระนาบพิกัด มีจุดอยู่ 2 จุด โดยแต่ละจุดแทนด้วยพิกัด `(x, y)`

ให้รับพิกัดของจุดทั้งสอง แล้วคำนวณระยะทางระหว่างจุดทั้งสองใน 2 รูปแบบ คือ **Euclidean Distance** และ **Manhattan Distance**

## คำอธิบายระยะทาง

**Euclidean Distance** คือระยะทางเส้นตรงที่สั้นที่สุดระหว่างจุดสองจุด คล้ายกับการลากเส้นตรงจากจุดหนึ่งไปยังอีกจุดหนึ่ง

สูตรคือ

```text
sqrt((x2 - x1)^2 + (y2 - y1)^2)
```

**Manhattan Distance** คือระยะทางที่เดินตามแนวแกนเท่านั้น โดยเดินตามแกน `x` และแกน `y` ไม่สามารถเดินเฉียงได้

สูตรคือ

```text
|x2 - x1| + |y2 - y1|
```

ตัวอย่างเช่น จุด `(0, 0)` และ `(3, 4)`

Euclidean Distance:

```text
sqrt((3 - 0)^2 + (4 - 0)^2)
= sqrt(9 + 16)
= sqrt(25)
= 5
```

Manhattan Distance:

```text
|3 - 0| + |4 - 0|
= 3 + 4
= 7
```

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `x1` และ `y1` แทนพิกัดของจุดที่ 1 |
| บรรทัดที่ 2 | จำนวนเต็ม `x2` และ `y2` แทนพิกัดของจุดที่ 2 |

โดยที่ `-1000 <= x1, y1, x2, y2 <= 1000`

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | แสดงระยะทางแบบ Euclidean ตามรูปแบบ `Euclidean: d` โดย `d` เป็นทศนิยม 2 ตำแหน่ง |
| บรรทัดที่ 2 | แสดงระยะทางแบบ Manhattan ตามรูปแบบ `Manhattan: m` โดย `m` เป็นจำนวนเต็ม |

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
|--------------------------|--------------------------|
| 0 0<br>3 4 | Euclidean: 5.00<br>Manhattan: 7 |
| -1 2<br>2 -2 | Euclidean: 5.00<br>Manhattan: 7 |
| 5 5<br>5 5 | Euclidean: 0.00<br>Manhattan: 0 |
