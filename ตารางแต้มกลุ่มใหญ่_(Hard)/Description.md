# ตารางแต้มกลุ่มใหญ่_(Hard)

ในรอบแบ่งกลุ่มของฟุตบอลโลก มีทีมหมายเลข 1 ถึง `n` แข่งขันกันหลายคู่ แต่ละคู่มีผลประตูของทั้งสองทีม

ทีมที่ชนะได้ 3 แต้ม ทีมที่เสมอได้ 1 แต้ม และทีมที่แพ้ไม่ได้แต้ม หลังจบการแข่งขันทั้งหมด ให้หาอันดับ 1 ของกลุ่มโดยใช้เกณฑ์ตามลำดับดังนี้

1. แต้มรวมมากกว่า
2. ถ้าแต้มเท่ากัน ให้ดูผลต่างประตูมากกว่า
3. ถ้ายังเท่ากัน ให้ดูประตูได้มากกว่า
4. ถ้ายังเท่ากัน ให้เลือกทีมหมายเลขน้อยกว่า

ให้แสดงทีมอันดับ 1 พร้อมแต้มรวม ผลต่างประตู และประตูได้

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `n` และ `m` แทนจำนวนทีมและจำนวนการแข่งขัน โดยที่ 2 <= n <= 10 และ 0 <= m <= 100 |
| บรรทัดที่ 2 ถึง m + 1 | จำนวนเต็ม `a b ga gb` แทนทีม `a` พบทีม `b` โดยทีม `a` ยิงได้ `ga` ประตู และทีม `b` ยิงได้ `gb` ประตู |

กำหนดให้ `1 <= a, b <= n`, `a != b` และ `0 <= ga, gb <= 20`

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | ทีมอันดับ 1 ตามรูปแบบ `Team x` |
| บรรทัดที่ 2 | แต้มรวม ตามรูปแบบ `Points: p` |
| บรรทัดที่ 3 | ผลต่างประตู ตามรูปแบบ `GD: g` |
| บรรทัดที่ 4 | ประตูได้ ตามรูปแบบ `GF: f` |
