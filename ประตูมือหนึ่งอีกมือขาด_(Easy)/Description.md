# ประตูมือหนึ่งอีกมือขาด\_(Easy)

ทางด้าน G. Donnaruma หลังจากชวดมาบอลโลกตลอดจนถึงปีนี้ เขาแค้นทีมชาติอิตาลีของเขามากเลยต้องการทราบว่าใครเป็นผู้ได้รับรางวัลถุงมือทองคำ

โดย Donnaruma ต้องการระบบคำนวณหาผู้รักษาประตูที่เก็บ Clean Sheet ตลอดการแข่งขันได้ (Clean Sheet คือการที่ผู้รักษาประตูไม่เสียประตูเลย หรือทีมตรงข้ามไม่สามารถทำประตูได้)

## ข้อมูลนำเข้า (Input)

| **ลำดับ**   | **รายละเอียด**                                                                |
| ----------- | ----------------------------------------------------------------------------- |
| บรรทัดที่ 1 | จำนวนเต็ม `g` แทนจำนวนประตูที่ผู้รักษาประตูโดนทำคะแนนใส่ โดยที่ 0 <= g <= 100 |

## ข้อมูลส่งออก (Output)

| **ลำดับ**   | **รายละเอียด**                      |
| ----------- | ----------------------------------- |
| บรรทัดที่ 1 | สถานะการเก็บ Clean Sheet (0 หรือ 1) |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
| ------------------------ | ------------------------ |
| 0                        | 1                        |
| 5                        | 0                        |
