# ห้องVAR\_(Easy)

ทาง FIFA รู้สึกว่าระบบ VAR อันเดิมใช้งานได้ไม่ค่อยถนัดเลยขอให้พวกเราสร้างระบบ VAR สำหรับเช็คประตูใหม่ให้กับเขา

โดยระบบจะทำการตรวตสอบว่าตำแหน่งของลูกบอล หากมากกว่าเส้นประตูให้ถือว่าเป็นประตู

## ข้อมูลนำเข้า (Input)

| **ลำดับ**   | **รายละเอียด**                                           |
| ----------- | -------------------------------------------------------- |
| บรรทัดที่ 1 | จำนวนเต็ม `b` แทนตำแหน่งลูกฟุตบอล โดยที่ 0 <= b <= 10000 |
| บรรทัดที่ 2 | จำนวนเต็ม `g` แทนตำแหน่งเส้นประตู โดยที่ 0 <= g <= 10000 |

## ข้อมูลส่งออก (Output)

| **ลำดับ**   | **รายละเอียด**               |
| ----------- | ---------------------------- |
| บรรทัดที่ 1 | สถานะการเป็นประตู (0 หรือ 1) |

## ข้อมูลนำเข้าและส่งออก

| **ตัวอย่างข้อมูลนำเข้า** | **ตัวอย่างข้อมูลส่งออก** |
| ------------------------ | ------------------------ |
| 105<br>100               | 1                        |
