# นักแยกจำนวนพิเศษ_(Medium)

ห้องทดลองของ ModCom ต้องการโปรแกรมช่วยแยกประเภทของจำนวนเต็ม โดยดูจาก "ผลรวมของตัวหารแท้" ของจำนวนนั้น

ตัวหารแท้ของจำนวน `x` คือจำนวนเต็มบวกที่หาร `x` ลงตัว แต่ไม่รวมตัว `x` เอง

ให้เขียนโปรแกรมรับจำนวนเต็มหลายจำนวน แล้วแสดงผลรวมของตัวหารแท้และประเภทของแต่ละจำนวน พร้อมสรุปจำนวนแต่ละประเภททั้งหมด

โจทย์นี้ต้องการฝึกการแยกงานเป็นฟังก์ชัน เช่น ฟังก์ชันหาผลรวมตัวหารแท้ และฟังก์ชันจัดประเภทของจำนวน

ประเภทของจำนวนมีดังนี้

| **เงื่อนไข** | **ประเภท** |
|-------------|------------|
| ผลรวมตัวหารแท้เท่ากับจำนวนนั้น | `PERFECT` |
| ผลรวมตัวหารแท้มากกว่าจำนวนนั้น | `ABUNDANT` |
| ผลรวมตัวหารแท้น้อยกว่าจำนวนนั้น | `DEFICIENT` |

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `n` แทนจำนวนตัวเลขที่ต้องตรวจสอบ โดยที่ 1 &le; n &le; 100 |
| บรรทัดที่ 2 ถึง n + 1 | จำนวนเต็ม `x` ที่ต้องตรวจสอบ โดยที่ 1 &le; x &le; 100,000 |

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 ถึง n | แสดงจำนวนเดิม ผลรวมตัวหารแท้ และประเภทของจำนวนนั้น ตามรูปแบบ `x sum TYPE` |
| บรรทัดที่ n + 1 | แสดงจำนวนประเภท `PERFECT` ตามรูปแบบ `Perfect: a` |
| บรรทัดที่ n + 2 | แสดงจำนวนประเภท `ABUNDANT` ตามรูปแบบ `Abundant: b` |
| บรรทัดที่ n + 3 | แสดงจำนวนประเภท `DEFICIENT` ตามรูปแบบ `Deficient: c` |
