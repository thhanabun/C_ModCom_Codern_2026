# เวลาการแข่งขันไม่ชนกัน_(Medium)

ฝ่ายจัดการแข่งขันฟุตบอลโลกต้องตรวจตารางการใช้สนามซ้อมหนึ่งสนาม แต่ละรายการมีเวลาเริ่มและเวลาจบเป็นนาที

ถ้าสองรายการมีช่วงเวลาซ้อนกัน จะถือว่าเป็นคู่เวลาที่ชนกัน โดยช่วง `[start, end)` หมายถึงเริ่มที่ `start` และจบก่อน `end` ดังนั้นรายการที่จบเวลาเดียวกับอีกรายการเริ่ม ไม่ถือว่าชนกัน

ให้รับตารางทั้งหมด แล้วแสดงจำนวนคู่เวลาที่ชนกัน

## ข้อมูลนำเข้า (Input)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนเต็ม `n` แทนจำนวนรายการ โดยที่ 1 <= n <= 100 |
| บรรทัดที่ 2 ถึง n + 1 | จำนวนเต็ม `start end` แทนเวลาเริ่มและเวลาจบของแต่ละรายการ โดยที่ 0 <= start < end <= 2000 |

## ข้อมูลส่งออก (Output)

| **ลำดับ** | **รายละเอียด** |
|----------|----------------|
| บรรทัดที่ 1 | จำนวนคู่เวลาที่ชนกัน |
